SDX Extension Config

Please configure the tags you wish to appear and how they should be displayed

Take a look at the readme for more info

Quick how-to

  • To add an alias, paste an emoji (or text) into the left box, and the slug of the tag in the right box
  • To find the slug of the tag, open the tags page in shibbydex and look at the end of the url for the slug, ie: https://shibbydex.com/tag/any-body-parts (the slug here is any-body-parts)
  • To delete a config item, just delete the text in either the alias or slug field and then hit save

You will need to hit save and then refresh shibbydex for changes made here to take effect

Please don't forget to click save!
